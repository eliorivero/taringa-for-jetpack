Extends Jetpack's Sharing feature adding a button to share a post to Taringa!

== Description ==

This is an extension for the Jetpack plugin: it extends the Sharing allowing to share a post, page, product or any custom post type to Taringa!

[Jetpack](https://jetpack.com/) is a a popular WordPress plugin which allows your site to include Sharing buttons, and lots of other cools features.

[Taringa!](http://www.taringa.net/) is the biggest Social Network in Latin America with more than 60M monthly unique visitors.

This project was developed during the [Media Party 2016](http://mediaparty.info/) Hackaton in Buenos Aires, Argentina.

Thanks to [Gino Cingolani](https://twitter.com/ginocingolani) and [Nicolás Cohen](https://twitter.com/cubetto), product manager and CTO of Taringa! respectively. Thanks to [Adriel Zárate](https://twitter.com/adriel_zarate) for the help with the icons.

== Installation ==

1. [Install Jetpack from their site](https://jetpack.com/install?from=wporg) by typing in your site address.

Alternatively install Jetpack via the plugin directory, or by uploading the files manually to your server. After activating Jetpack, connect to WordPress.com to enable the Jetpack features.

If you need additional help [read our detailed instructions (including a video walkthrough)](http://jetpack.com/support/installing-jetpack/).

2. Activate the Sharing module.

3. Install this plugin. You can search for it in the WP Admin area going to Plugins > Add New.

4. Once it's installed, activate it and go to Settings > Sharing to configure the button.
